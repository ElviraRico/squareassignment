Semantic Personal Publishing Platform

Installation: BD

  1. Download the attached database email.
  2. Create a new DB with utf8_general_ci collation in your phpmyadmin local server.
  3. Import the attached database email.

Installation: Wordpress

  1. Unzip the package with wp documents in a local sever like xampp.
  2. Open wp-config.php with a text editor and fill in your database connection details. Change:
    1. The name of the database.
    2. MySQL database username.
    3. MySQL database password.

Wordpress admin
